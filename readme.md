## 编译

```
npm run compile
npm run compile:w  // 可监听
```

## 测试

```
npm run test
```

## 打包

```
npm run build
```

## 自定义打包

```
// 打包umd格式，兼容浏览器和node环境
npm run dev

// 打包cjs格式
npm run dev:cjs

// 打包es6格式
npm run dev:esm
```

## 发布
```
npm publish
```
